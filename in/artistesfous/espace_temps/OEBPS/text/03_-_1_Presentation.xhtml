Marie Latour

God save America

Une machine à voyager dans le temps est un outil à ne pas mettre entre toutes les mains, surtout quand le monde s’apprête à basculer dans le fascisme. Mais quelles sont les bonnes mains ? Existe-t-il de telles mains ? Marie Latour, artiste folle des premiers jours, vous invite dans ce thriller, de version modifiée en version modifiée de la réalité ; une course en avant qui sera lourde de conséquences pour les protagonistes comme pour le monde qui les entoure.
